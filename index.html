Hello, I'm Jorge!


I’m a full stack developer working in the NYC metro area. I like writing code, but I especially like writing lines of code that become pretty visuals or cool interfaces to end users. I’ve had a a passion for front end since my college days when browsing CSS Zen Garden.

Give me an application idea and I can spend hours daydreaming about how it’ll look. I enjoy getting lost in the process of translating design wireframes/mocks to actual code that will produce visual elements; I’m passionate about combining the art of design with the art of programming in order to execute and deliver elegant products and interfaces. My goal is creating beautiful yet functional interfaces.

Besides daydreaming about front end stuff, I’m a fitness enthusiast and a cake lover (a lover of anything with the word “cake” in it to be more precise... pancakes, cupcakes, etc.); you might see this reflected in my work. Also, I’m a newbie skiing addict, so don’t hesitate to offer help if you see me buried under the snow on the slopes ☺.

Cupcake Finder

Single Page Application/Design Patterns/Responsive
OVERVIEW

This a single page application featuring locations of cupcake shops near you built using a Model-View-ViewModel pattern. It allows the user to add/remove shops from his/her Favorite list and retrieve details from a shop as contact information, reviews and photos. Knockout.js, Materialize, Google Map API, Foursquare API. github/get-your-cupcakes

My Food Journal

Design PAtterns/model–view–presenter/Responsive
OVERVIEW

Single page app that tracks the user's calorie intake, and other health-related metrics. Users are able to search for food items from the database provided by the health API and add these items to one or several meals for macro-nutrient intake tracking. Backbone,js, Semantic-UI, Nutritionix API. github/my-food-journal
